\section{Structural Characterisations}
\label{sec:interpreting-paths}

In this section, we will be interested in extracting a very regular structure
from a graph class $\mathcal{C}$ that is not \kl{$\forall$-well-quasi-ordered}.
This structure will take two forms, the first one is a combinatorial notion of
\kl{periodic antichain} that is similar to the one proposed in \cite[Section
7]{ALM17}, and the second one is a logical property of \kl{existentially
transducing} the class of all finite paths.
\maelin{Three things, periodic sequence of graphs forming a antichain, we show that in our setting this is equivalent to periodic antichain as defined \cite[Section
	7]{ALM17}. We  will show that we can \kl{existentially
		transducing} the class of all finite paths from these sequences.}


\begin{definition}[Periodic Sequence]
  A \intro{periodic sequence (of graphs)} is given by a finite graph $G$,
  a labelling function $\lab \colon V(G) \to \Sigma$ for some finite set
  $\Sigma$, and two sets $C, F \subseteq \Sigma^2$. It defines
  an infinite set of graphs $\seqof{G^r}[r \geq 1]$ as follows:
  the vertex set of $G^r$ is $V(G) \times \set{1, \ldots, r}$, and there is an edge
  between $(u,i)$ and $(v,j)$ with $i \leq j$ if and only if one of the following holds:
  \begin{itemize}
      \item $(u,v) \in E(G)$ and $i = j$;
      \item $(\lab(u), \lab(v)) \in C$ and $|i - j| = 1$
      \item $(\lab(u), \lab(v)) \in F$ and $|i - j| > 1$.
  \end{itemize}
\end{definition}


It follows from results of the previous section that one can extract
\kl{periodic sequence} forming an antichain from any hereditary class of graphs that is not
\kl{$\forall$-well-quasi-ordered}.

\begin{lemma}
  \label{lem:regular-antichain}
  If a hereditary class of graphs $\Cls$ is not \kl{$\forall$-well-quasi-ordered},
  then it contains a \kl{periodic sequence} $\seqof{G^r}[r \geq 1]$ that is an
  \kl{antichain} for the \kl{labelled induced subgraph} relation.
\end{lemma}
\begin{proof}
  Using 
  \cref{cor:hereditary-perfect-boughs-wqo}, one can assume that $\Cls$
  has \kl{bounded linear clique-width}. It then follows from
  the analysis of \cite{LOPEZ24} that 
  $\Cls$ contains a \kl{periodic sequence} $\seqof{G^r}[r \geq 1]$.
\end{proof}

For $i\leq r$, we denote the $i$-th copy of $G$ in $G^r$ by $G_i$, and by $u_i \in G^r$ the $i$-th copy of $u\in V(G)$. We will proceed in two steps, we first define a formula producing a directed graph where there is a directed shortest path from $G_1$ to $G_r$ going through every $G_i$, but there may exist arcs going from $G_i$ to $G_j$ for $j\leq i$. We will then show how to handle theses backward arcs, and produce a path $P_{r-2}$.

Define the following formula:
$$\phi_{\rightarrow}(u,v) = \lnot ((u,v) \in E(G^r) \Leftrightarrow (l(u),l(v)) \in F) $$
Notice that this formula is quantifier-free. Informally, $\phi_{\rightarrow}(u,v)$ creates an arc $(u,v)$ if the edge relation between $u,v \in G^r$ differs if $u$ and $v$ are put far away form each other in this order (note that $u$ can be in a copy of $G$ that comes after the one of $v$ in $G^r$).
Let $H^r = \someInterp(G^r)$ for $\someInterp = (\top, \phi_{\rightarrow}, \top)$, note that $H$ is a directed graph as $\phi_{\rightarrow}$ is not symmetric. 
First, observe that in $H^r$ there is no arc between two non-consecutive copies.  

\begin{claim}\label{clm:nolongforwardarc}
	For $(u_i,v_j) \in E(H^r)$ only if $j\leq i+1$.
\end{claim}
\begin{claimproof}
	If $j \geq i+1$, then $uv \in E(G^r)$ if and only if $(l(u),l(v)) \in F$ by definition of $G^r$.
\end{claimproof}

From definition, it is clear that if there is a backward arc from the $i$-th copy to the $j$-th copy with $j<i$, then for every other previous copy, this arc also exist.

\begin{claim}\label{clm:regularbackward}
	If $(u_i,v_j) \in E(H^r)$ for $j<i$ then for every $j'\leq i$, $(u_i,v_{j'}) \in E(H^r)$.
\end{claim}

A spanning path of $H^r$ is a directed shortest path from $V(H_1)$ to $V(H_r)$ going through at least one vertex from each $V(H_i)$.

\begin{claim} \label{clm:spanningpath}
	There is a spanning path in $H^r$.
\end{claim}
\begin{claimproof}

	We now argue that if there is no spanning path, then one can embed $G^r$ into $G^{r+2}$. Fix $1<i<r$, and define the mapping $h$ as follows: map the first $i-1$ copies of $G$ in $G^r$ and $V_i^+$ and $V_i^o$ to the corresponding first $i$ copies in $G^{r+2}$; and to the $r-i$ $i+2$-th copy of $G$ in $G^{r+2}$; finally, $V_i^-$ is mapped to the $i+2$-th copy of $G$ in $G^{r+2}$, and the last copies of $G$ in $G^r$ are mapped to the last copies of $G$ in $G^{r+2}$.
	It is clear from construction that the $i+1$-th copy of $G$ in $G^r$ is left untouched by this mapping and that the first. 
	Suppose now that this is not a correct embedding, then it means that there is $u \in V_i^+ \cup V_i^o$ and $v\in V_i^-$ such that $(u,v) \in E(G^r)$ and $(h(u),h(v)) \notin E(G^{r+2})$ or the reverse. 
	However as $(h(u),h(v))$ is entirely defined by $(\lab(u),\lab(v))$ and $F$, it means that $\phi_{\rightarrow}(u,v)$ is satisfied and $(u,v)\in E(H^r)$.
	Hence if there is no spanning path, then $(G^r)_{r\geq 1}$ is not an antichain.
\end{claimproof}

Finally we provide a last structural characterisation defined in \cite[Section 7]{ALM17} that is not \kl{$\forall$-well-quasi-ordered} similar to periodic sequence of graphs. 
In : given a finite alphabet $\Sigma$, and 
a finite set $S \subseteq \Sigma^2$, one can map a word 
$w \in \Sigma^*$ to a graph $G_w$ whose vertex set is
the positions of $w$, and where there is an edge between two positions
$i + 1 < j$ if and only if $(w[i], w[j]) \in F$,
and an edge between two consecutive positions $i$ and $i + 1$
if and only if $(w[i], w[i + 1]) \not\in F$.  In this setting, a \intro{periodic sequence} is given by a finite word $w
\in \Sigma^*$, and a finite set $F \subseteq \Sigma^2$, and defines the
infinite set of graphs $\seqof{G_{w^r}}[r \geq 1]$.
It is easy to see that every \kl{periodic antichain} is also a
\kl{periodic sequence of graph} as defined in this paper, by grouping the vertices of $w$ into a single graph $G$ and adapting colours.

\begin{corollary}
	\label{cor:periodicgraph_WPA}
	Let $\Cls$ be a hereditary class, then $\Cls$ contains a periodic sequence of graph forming an antichain if and only if it contains a weak periodic antichain.
\end{corollary}
\begin{proof}
	Observe that the sequence of vertices obtained from spanning path \cref{clm:spanningpath} correspond  to a periodic antichain, with $\Sigma$ being the labels used on vertices of the path, and $F$ defined according to $G$.
\end{proof}
